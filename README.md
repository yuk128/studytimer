# study-timer

공부 타이머

안드로이드 앱 실습 기말 구상

## 기능
### 필수
타이머 작동, 중지, 삭제
집중시간 기록(외부 시간 불러오기 필수)
유저가 시간 조정

### --필요 지식
타이머 구현
외부시간 불러오기
유저와 상호작용해서 변수 조정

선택(이지만 필수에 가까운 권장) 집중 시간과 휴식 시간 구분 중지상태에서 집중타이머는 멈추게 휴식타이머는 안 멈추게 원형 시간 바 데이터 저장 --필요지식 중지상태 호출 다른건 지식보단 아이디어가 필요 데이터저장 및 불러오기(room사용, 난이도 상 예상)

선택(약한권장) 간단한 todo리스트(집중기간동안 뭐했는지) 화면이 꺼지는건 집중타이머가 멈추지 않음 시간 기록을 타임라인으로 색별로 기록 --필요지식 중지상태와 화면 꺼짐 상태 구분(난이도 중상 예상) 타임라인은 아이디어 필요(난이도 중상 예상) 데이터를 어딘가에 저장하고 불러오기(난이도 상 예상)

선택(안해도됨) 스톱워치 모드(타이머가 아니라 시간만 재는 용도(자유로 휴식 집중 껏다 키는) 하루 총 집중시간이나 노는시간 측정해서 몇시간인지 적어두기 달력 형으로 정리 --필요지식 없을걸로 예상 지식보단 아이디어가 필요

계획(4주, 28일)
1주차
할일: 계획및 아이디어 정리(같이 개발할 방법정리도)
예은: 계획및 구상
소영: 아이디어및 기능 생각

2주차
할일: ui디자인 완성및 필수 기능 개발 (이떄 제출 가능 상태 만들어야 함)
역할1: ui구상/구현, 집중시간 기록(외부시간 불러오기)
역할2: 타이머 제작, 데이터 저장(조정필요)

3주차
항일: 집중 휴식 구분 및 정지설정, 원형시간바, 4주차 ui구상(4주차도 어느정도 진행해야함)
역할1: 원형시간바, 4주차 ui구상
역할2: 집중 휴식 구분및 정지 설정

4주차
할일: 화면꺼진상태와 정지상태 구분, 그 외 선택 사항 구현(시간 남으면 기타 선택사항 전부구현 및 버그 수정)
역할분배 없음 그때봐서 유동적으로

5주차
할일: 있으면 안됨, 버그 수정및 최종 점검 역할 분배 없음
