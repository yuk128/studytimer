# study-timer

앱 개발 내용

아래는 코드에 적힌 순서대로 프로그램에 대한 설명을 적었습니다.

1) StudyTimerApp()
    - 우선 StudyTimerApp()함수에서 대부분의 타이머 기능을 만들었습니다.
    - 상단에 공부시간과 휴식시간에 대해 입력받은 시간을 getFocusSeconds(), getRestSeconds() 두 함수를 통해 초로 환산했습니다.
    - makeRecord()는 공부시간에 타이머가 끝나거나 중단되면 해당 시간을 기록화면에 입력합니다.
    - 이어지는 LaunchedEffect()에서 본격적으로 타이머를 다루게 됩니다.
    - 상단에서 isRunning라는 변수를 선언했습니다. isRunning가 true면 LaunchedEffect가 작동하고 while (isRunning)은 타이머가 작동중인지 파악합니다.
    - 1초 마다 하단의 if문이 작동하는데 이 부분은 시간이 남아있는지 남아 있다면 시간을 줄이고 남아있지 않다면 다음 타이머로 넘어가거나 멈추도록 합니다.
    - 그 다음엔 Column이 나오는데 여기서는 각각 기록화면과 타이머화면, todo리스트 화면을 전환 할 수 있게 해줍니다. 또한 그 하단에는 화면을 전환 할 수 있는 이미지 버튼이 존재 합니다.
2) TimerScreen()
    - DisposableEffect()에서는 화면 이탈시 타이머가 멈추도록 합니다. 그래서 다른 앱을 실행 중일땐 공부시간으로 기록 되지 않습니다.
    - 이어지는 Column()에서는 타이머 작동 여부에 따라 시간이 뜨거나 시간을 입력 받는 창이 뜨게 됨니다. 또한 상단에 책아이콘이나 게임기 아이콘이 뜨게 만들기도 합니다.
    - Box를 선언하고 CircularTimer(아래서 작성)에서 원형시간바를 그린후 Column()을 이용하여 중앙에 내용을 적습니다. 타이머가 작동중일땐 잔여시간을 아닐때는 시간입력창 6개를 보여줍니다.
    - 그 다음에는 Row()를 이용하여 리셋, 시작, 중단 버튼을 배치 하였습니다. SquareButton()은 이 세 버튼을 디자인 하기 위한 용도입니다.
    - 위에서 사용한 CircularTimer()가 위치한 곳이다. 아래에선 각각 공부 시간과 휴식 시간에 따라 색을 다르게 보여주고 전체 시간과 남은 시간을 계산하여 표시합니다.
3) RecordScreen()
    - Column()이 바로 나오며 화면을 구상하고 삭제 버튼과 수정버튼은 IconButton로 안드로이드스튜디오에 기본으로 있는 리소스를 활용하였습니다.
    - AlertDialog()가 삭제 혹은 수정 버튼을 눌렀을 때 각각 삭제할지 물어보는 창, 어떻게 수정할지 입력받는 창을 띄워 줍니다.
4) ProfileScreen()
    - 프로필 영역은 3구역으로 나눠서 상단에는 간단한 사진과 이름이, 중간에는 다짐과 같은 말을 적을 수 있는 칸이, 화면의 절반 이상을 차지하는 하단에는 간단한todo리스트를 만들었습니다.
    - 상단 영역에는 단순히 날짜와 이름, 사진을 넣었습니다.
    - 중간부분은 PledgeBox()로 글을 적어 넣을 수 있게 했으며 우측에 보이는 수정버튼 역시 기록화면에서 사용한 것과 동일한 것 입니다.
    - PledgeBox()에서는 연필 아이콘을 클릭하면 창을 띄워 적고싶은 말을 입력 받게 했습니다.
    - 하단에는 todo리스트로 Column대신 LazyColumn을 사용하여 리스트 양이 많아지면 스크롤로 볼 수 있게 하였습니다.
